# Automatizace Cookie Clicker

## Popis a cíl projektu
Tento projekt je Python skript pro automatizaci klikání ve hře Cookie Clicker pomocí knihovny `pyautogui`. Cílem je automatizovat generování cookies pro zábavu a testování automatizace.

## Funkcionalita programu
Program obsahuje dva současné smyčky: jednu pro automatické klikání myší a druhou pro sledování klávesnice. Používá threading pro současné běh. Klávesa '8' přepíná klikání, '7' ukončí program. Spouští se po 5 sekundách pro přepnutí do okna hry.

### Technická část
- Použité knihovny: `pyautogui` (pro ovládání myši), `keyboard` (pro detekci kláves), `threading` (pro současné smyčky).
- Algoritmy: Jednoduché smyčky s časovými prodlevami pro kontrolu stavu.
- Žádné externí API nebo vlastní datové struktury.
